# Aurion Clinical AI — Design System

Premium, clinical-grade, calm and precise. Light mode. Generous whitespace. Never flashy.
Typeface: **Inter**. Reads as the same product as the Aurion iOS app.

## Brand
- Logo: existing Aurion mark — a navy rounded-square (squircle) icon with a gold "A" + comet/star.
  Treat as a fixed asset; never redesign, recolor, or substitute it.

## Colors
- Primary / brand navy: `#0C1B37`
- Navy gradient stops: `#16284E`, `#081226`
- Accent (default) gold: `#C9A84C` (light `#E5D082`, dark `#B5953D`)
- Background / canvas: `#F5F6FA`
- Surface / card: `#FFFFFF`
- Border / hairline: `#E6E9EE`
- Text primary: `#0C1B37`
- Text secondary: `#6B7280`
- Success (fixed): `#2E9E6A`
- Warning / conflicts (fixed): `#D9941F`
- Error (fixed): `#D9352B`
- Info (fixed): `#2D6CDF`

## Typography (Inter)
- Large title 34px / 700, tight tracking
- Title 22px / 600
- Title-3 20px / 600
- Headline 17px / 600
- Body 17px / 400
- Callout 15px / 500
- Caption 13px / 400
- Eyebrow 11px / 600, UPPERCASE, letter-spacing

## Shape & elevation
- Radius: card 16px, button 12px, chip 10px, modal/sheet 20px, small 6px
- Card shadow: soft 2-layer (`0 1px 2px rgba(12,27,55,.04), 0 6px 18px -6px rgba(12,27,55,.08)`)
- Primary button carries a subtle gold glow

## Components
- Primary button: gold fill, navy text, gold glow
- Secondary button: white fill, navy text, hairline border
- Status badge: pill, semantic-tinted background + text (success/warning/error/info)
- Card: white surface on canvas, 16px radius, soft shadow
- Left sidebar nav: navy logo chip at top, line icons, gold active state with left indicator
- Tables: quiet hairline rows, sticky header, hover tint, mono code chips for IDs
- Motion: 320ms cubic-bezier(0.32, 0.72, 0, 1) fade / slide-up
