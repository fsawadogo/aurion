## Operational notes
- Backend stack came up via `docker-compose up -d --no-deps aurion-api` because LocalStack v2026 (`localstack/localstack:latest`) requires a `LOCALSTACK_AUTH_TOKEN` even for community tier — license activation fails (exit 55) without one. FastAPI runs fine without it; only AWS-emulating endpoints (S3/DynamoDB upload paths) would fail at runtime.
- The first PR (#109) was merged via `gh pr merge --auto` and went immediately because the repo lacks branch-protection required checks. Subsequent PRs (#111, #112) explicitly waited on `gh pr checks --watch` before merging — closer to the workflow's "auto-merge on green CI only" intent. **Recommend** enabling required-checks on `main` via repo settings so `--auto` semantics match the workflow spec.
- macOS-sync junk duplicates (`config 2.py`, `service_discovery 2.tf`, etc.) caused the migration test suite to fail until removed — a stray `2026_05_28_0006_provider_overrides 2.py` made Alembic see revision `0006` twice. Untracked files; deleted locally with `rm`.
