## Operational notes
- Backend stack came up via `docker-compose up -d --no-deps aurion-api` because LocalStack v2026 (`localstack/localstack:latest`) requires a `LOCALSTACK_AUTH_TOKEN` even for community tier ‚Äî license activation fails (exit 55) without one. FastAPI runs fine without it; only AWS-emulating endpoints (S3/DynamoDB upload paths) would fail at runtime.

Subsequent PRs (#111, #112) explicitly waited on `gh pr checks --watch` before merging ‚Äî closer to the workflow's "auto-merge on green CI only" intent. **Recommend** enabling required-checks on `main` via repo settings so `--auto` semantics match the workflow spec.
